## URL形式
GitHub Pagesのリポジトリ名を `yurukan-public-assets` にする場合、画像URLは次の形式になります。

```text
https://<github-user>.github.io/yurukan-public-assets/x-posts/2026/06/x_post_2026-06-21_12-00_soccer.png
```

`<github-user>` はGitHubユーザー名に置き換えます。

## 注意
- このフォルダの画像は公開前提です。
- GitHub Pagesに置いた画像は、削除してもGit履歴やキャッシュに残る可能性があります。
- 秘密情報、個人情報、投稿前に絶対見られたくない画像は置きません。
